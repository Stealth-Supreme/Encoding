# Encoding

<h3>This is a simple Python program that encodes and decodes text using a custom algorithm.</h3>
<hr>
Overview<br>
The program lets you convert normal text into an encoded format and also decode it back to the original text.
<br><br>
Usage<br>
To use the program, call the function with your input string.
<br><br>
Example<br>
coding("I like Stealth-Supreme")
<br><br>
Output<br>
umA qaammllUpvRo ttKAhgvRoUAIqaaAhgwwL-ttKCPxbOsnRQvRoQapvRo
